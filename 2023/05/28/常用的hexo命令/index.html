1.端口

  • 检查4000端口是否有被占用
    netstat -o -n -a | findstr :4000
  • 结束端口进程
    taskkill /F /PID 3628

    ‘3628’处为最末尾数字

2.连接

ssh -T git@github.com

3.常用的hexo命令

  • 启用本地服务器，并预览主题
    hexo s

*清除缓存文件和已生成的静态文件
hexo clean

  • 生成网站静态文件到默认设置的public文件夹
    hexo g

  • 生成网站静态文件，并部署到设定的仓库
    hexo d
